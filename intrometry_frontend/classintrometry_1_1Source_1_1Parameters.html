bool intrometry::Source::Parameters::persistent_structure_

If true assume that the number and order of entries is preserved, if false new names are generated and published on each write (it is unlikely that you want this). It is not possible to detect persistent structure automatically, since variable containers may have constant size. Wrong setting is not going to result in an error, but metric IDs and values may not match.
